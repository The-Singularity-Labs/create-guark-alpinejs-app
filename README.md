# Guark Vue Template
Guark Alpine.js app template.

![Guark Alpine Template](https://i.imgur.com/RhU6bh7.png)

## Install

Create new empty directory and cd to it, and run:
```bash
git clone 
```

## Run dev app.

Run dev app with vue hot reload:
```bash
guark run
```

## Build

```bash
guark build
```

## Bundle

```bash
guark bundle
```
